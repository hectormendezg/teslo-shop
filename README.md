# Teslo API

1. Clonar proyecto
2. Ejecutar

```
yarn install
```

3. Renombrar archivo `.env.template`a `.env`
4. Completar las variables de entorno
5. Crear la base de datos

```
docker-compose up -d
```

6. Ejecutar SEED

```
http://localhost:3000/api/seed
```

7. Ejecutar en desarrollo

```
yarn start:dev
```
